# react-native-threed-viewer

3D viewer for react-native

## Installation

```sh
npm install react-native-threed-viewer
```

## Usage

```js
import { ThreedViewerView } from "react-native-threed-viewer";

// ...

<ThreedViewerView color="tomato" />
```
